comfy script

comfy is a cross-platform command script manager / runner tool, which allows you to run commands in the command line itself, but being these predefined in a portable and universal .comfy file.

.comfy files are plain text, but with some arguments so the comfy parser understands what do you want to do.

why I started comfy

I started comfy as a simple project to expand my knowledge about the Rust programming language. Anyway, it has become a quite usable and interesting project. You can check it out here.

termlock

termlock is a lock for your terminal written in Rust. It supports features such as a custom password and the ability to set a maximum number of wrong passwords before completely locking the terminal.

ricetools

ricetools is a tool (written in Rust) that allows you to generate small decorations for your rice or customization of a Linux distro. Decorate your screenshots with the most original elements ;-;

ricetools can generate:

  • Ghosts
  • Bunny
  • Pacman

build

In order to build termlock or ricetools you will need to have Rust (and optionally git) installed, then simply do:

  • git clone https://github.com/dacousb/termlock
  • cd termlock
  • cargo build --release
or
  • git clone https://github.com/dacousb/ricetools
  • cd ricetools
  • cargo build --release

brwrs

brwrs is a new protocol running over TCP/IP that is intended to be a suitable candidate for terminal-only servers (plain text data). That is, although it can be accessed from a browser, brwrs will not correctly interpret the browser's GET request.

brwrs uses a request protocol similar to HTTP(S), but aims to simplify it a bit, removing unnecessary elements such as the UserAgent and browser fingerprinting. By default, the server software does not log IPs, but that can be easily achieved modifying the source code.
